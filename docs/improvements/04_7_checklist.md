# 구현 완료 체크리스트
> 각 작업 완료 후 체크. 전체 통과해야 배포 가능.

---

## 토큰 & 글로벌 (04_1)

- [ ] 배경이 `#F0EEE9` 인가? (`#fff`, `#f5f5f5` 금지)
- [ ] 카드/벤토셀 배경이 `#E4E1DA` 인가?
- [ ] 모든 검정이 `#2a2620` 인가? (`#000`, `#111` 금지)
- [ ] 골드 텍스트가 `#7a5c20` 인가? (`#c8a96e`는 장식 전용)
- [ ] `font-weight`가 300/400/500/600 만 사용되는가? (700 금지)
- [ ] `box-shadow`가 FAB 버튼 외에 없는가?
- [ ] grain 텍스처가 `body::after`로 전역 적용되는가?

## 폰트 (04_1)

- [ ] 제목이 `Noto Serif KR 600` 인가?
- [ ] 본문이 `Noto Sans KR 400/300` 인가?
- [ ] 레이블·날짜가 `DM Mono` 인가?
- [ ] 한글 포함 버튼이 `'DM Mono', 'Noto Sans KR'` 혼합인가?
- [ ] `Space Mono`, `VT323`, `Bebas Neue`, `Pretendard` 가 없는가?
- [ ] 굴림체·궁서체·바탕체가 없는가?

## 바텀 내비 (04_2)

- [ ] 바텀 내비에 `backdrop-filter: blur(16px)` 있는가?
- [ ] FAB이 탭바 위로 16px 솟아있는가?
- [ ] FAB 클릭이 탭 전환이 아닌 `router.push`인가?
- [ ] 하루 현상에서 뒤로가기 시 이전 탭으로 복귀되는가?
- [ ] `safe-area-inset-bottom` 처리가 있는가?

## 홈 화면 (04_3)

- [ ] 벤토 그리드가 CSS Grid로 구현되었는가? (flex 금지)
- [ ] STREAK 셀이 `tall dark`인가?
- [ ] TODAY 셀이 `gold`인가?
- [ ] MOOD FLOW 셀이 `wide`인가?

## 채팅 (04_4)

- [ ] AI 말풍선 좌상단 radius가 `2px`인가?
- [ ] 유저 말풍선 우상단 radius가 `2px`인가?

## 필름롤 (04_5)

- [ ] 필름 카드 배경이 `#E4E1DA` (밝은 베이지)인가? (어두운 색 금지)
- [ ] 감정 컬러가 `.emotion-bar`에만 적용되는가?
- [ ] 스프로킷 구멍 색이 `var(--surface-base)`인가?

## 캘린더 (04_5)

- [ ] 오늘 날짜 원이 `#7a5c20`인가? (보라색 금지)
- [ ] 기록 있는 날 배경이 `var(--gold-pale)`인가?

## 로그인 화면

- [ ] 온보딩 필름스트립이 위아래 끝에 딱 붙어있는가?
- [ ] `safe-area-inset-top/bottom` 처리가 있는가?
